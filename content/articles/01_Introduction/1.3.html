1.3 - Mes premiers programmes

Introduction

Maintenant que tu sais ce qu'est le développement et que ton environnement est prêt, il est temps d’écrire tes premiers programmes ! Ce tutoriel te guide à travers la création de programmes très simples dans différents langages pour que tu comprennes comment ils fonctionnent.

1. L’objectif d’un premier programme

Le "Hello World" est le programme le plus simple qu'on écrit en général pour tester un langage de programmation. Il consiste simplement à afficher un message à l’écran. Cela te permet de vérifier que ton environnement fonctionne bien.

2. Écrire et exécuter un programme en Python

a. Code :

python
print("Hello, World!")

b. Étapes :

  1. Ouvre ton éditeur (ex. : VS Code)

  2. Crée un fichier : hello.py

  3. Colle le code ci-dessus

  4. Ouvre un terminal, et exécute :

    bash
    python hello.py

Résultat attendu :

Hello, World!

3. Écrire et exécuter un programme en JavaScript (Node.js)

a. Code :

javascript
console.log("Hello, World!");

b. Étapes :

  1. Crée un fichier : hello.js

  2. Colle le code

  3. Exécute avec Node :

    bash
    node hello.js

Résultat :

Hello, World!

4. Écrire et exécuter un programme en HTML + JavaScript

a. Code HTML :

html
<!DOCTYPE html>
<html>
<head>
  <title>Mon premier programme</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <script>
    console.log("Hello depuis JavaScript !");
  </script>
</body>
</html>

b. Étapes :

  1. Crée un fichier : index.html

  2. Ouvre-le dans un navigateur

  3. Regarde la page + la console du navigateur (F12 > onglet "Console")

5. Écrire et exécuter un programme en Java

a. Code :

java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

b. Étapes :

  1. Crée un fichier : Hello.java

  2. Compile le code :

    bash
    javac Hello.java

  3. Exécute :

    bash
    java Hello

6. Écrire et exécuter un programme en C

a. Code :

c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

b. Étapes :

  1. Crée un fichier : hello.c

  2. Compile :

    bash
    gcc hello.c -o hello

  3. Exécute :

    bash
    ./hello

7. Concepts de base communs

Concept Description
Affichage Commande pour afficher du texte à l’écran (ex. print, console.log)
Fichier source Le fichier qui contient ton code (.py, .js, .java, etc.)
Exécution / Compilation L’action de lancer ton programme pour qu’il s’exécute
Console La zone où tu vois le résultat ou les erreurs de ton programme

8. Choisir un langage pour commencer

Langage Facilité Cas d’usage
Python ★★★★★ Facile, lisible, très utilisé
JavaScript ★★★★☆ Web, interactions, côté client
Java ★★★☆☆ Applications robustes, Android
C ★★☆☆☆ Apprentissage bas niveau, systèmes

Pour débuter, Python ou JavaScript sont généralement recommandés.

9. Exercices pratiques

  1. Modifie ton programme pour qu’il affiche ton prénom.

  2. Affiche plusieurs lignes :

    python
    print("Bonjour")
print("Je m'appelle Alice")
print("J'apprends à coder")

  3. Teste les erreurs : supprime une parenthèse et regarde le message d’erreur.

  4. En JavaScript, affiche une addition :

    javascript
    console.log(2 + 3);

10. Conclusion

Félicitations 🎉 ! Tu viens d’écrire et d’exécuter tes premiers programmes. Ce n’est que le début. Tu vas maintenant apprendre à écrire des programmes plus complexes avec des conditions, des boucles, des fonctions et plus encore.

Points à retenir :

  • Chaque langage a sa propre syntaxe, mais les concepts sont souvent similaires.

  • Le programme “Hello World” est une bonne façon de commencer.

  • Utiliser la console est essentiel pour voir ce que fait ton code.
