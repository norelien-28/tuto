2.4.2 – Installation de React et structure de projet

⚙️ 1. Prérequis

Avant d’installer React, il te faut :

  • Node.js (version ≥ 14)
    👉 Télécharger Node.js

  • npm (fourni avec Node.js) ou yarn
    👉 Vérifie leur présence avec : 

    bash
    node -v
npm -v

  • Un éditeur de code (ex: VS Code)

🚀 2. Créer une application React

La méthode la plus simple pour démarrer avec React est d’utiliser l’outil officiel : Create React App (CRA).

🛠️ Installation avec npx (recommandé)

bash
npx create-react-app mon-projet-react

Remplace mon-projet-react par le nom de ton projet.

📁 3. Structure de base d’un projet React

Une fois le projet créé, tu obtiendras une arborescence similaire à :

php
mon-projet-react/
├── node_modules/         # Dépendances du projet
├── public/               # Fichiers statiques (HTML, favicons, images)
│   └── index.html        # Point d’entrée de l’app (HTML)
├── src/                  # Code source principal (JS/JSX, CSS)
│   ├── App.js            # Composant principal
│   ├── App.css           # Style associé à App.js
│   ├── index.js          # Point d’entrée JS (rendu React)
│   └── ...
├── package.json          # Dépendances et scripts du projet
├── .gitignore            # Fichiers ignorés par Git
└── README.md             # Documentation du projet

🧠 4. Fichiers clés à connaître

Fichier / Dossier Rôle
public/index.html Contient une <div id="root"> où l'app React sera injectée
src/index.js Monte l'app React dans le DOM avec ReactDOM.render() ou createRoot()
src/App.js Premier composant affiché à l'écran
package.json Gère les scripts (ex : start, build, test) et les dépendances

▶️ 5. Lancer l’application

Pour démarrer l'application localement :

bash
cd mon-projet-react
npm start

Cela ouvrira automatiquement http://localhost:3000 dans ton navigateur.

🧹 6. Nettoyage de base (facultatif)

Le projet CRA vient avec du contenu de démonstration. Tu peux le nettoyer :

  • Supprimer le logo et les styles inutiles.

  • Modifier App.js pour repartir de zéro.

Exemple minimal de App.js :

jsx
function App() {
  return (
    <div>
      <h1>Bonjour React !</h1>
    </div>
  );
}

export default App;

🔄 7. Scripts utiles

Dans package.json, tu trouveras ces scripts :

Script Utilité
npm start Démarre le serveur de développement
npm run build Crée une version de production
npm test Lance les tests
npm run eject Récupère la configuration Webpack (⚠️ irreversible)

💡 Astuce : Utiliser VS Code

  • Installe l’extension ESLint pour le linting.

  • Utilise Prettier pour un code formaté automatiquement.
