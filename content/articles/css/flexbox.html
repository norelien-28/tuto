Introduction à Flexbox

Flexbox est un module CSS puissant pour créer des mises en page flexibles et responsives.

Quelques propriétés clés :

  • display: flex; active le mode flex sur un conteneur.
  • justify-content aligne les éléments horizontalement.
  • align-items aligne les éléments verticalement.
  • flex-direction définit la direction des éléments (ligne ou colonne).

Exemple simple :


.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
}

Pour en savoir plus, consulte la documentation MDN Flexbox.
