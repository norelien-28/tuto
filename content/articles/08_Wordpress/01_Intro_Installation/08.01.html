8.1 – Introduction à WordPress

🧭 Qu’est-ce que WordPress ?

WordPress est un système de gestion de contenu (CMS) libre et open-source. Il permet de créer et gérer facilement un site web sans avoir besoin de tout coder à la main.

Il est utilisé aussi bien pour des blogs personnels, des sites vitrines, des portfolios, que des boutiques en ligne, voire même des applications web grâce à son API REST.

💡 Quelques chiffres

  • 🎯 +43% des sites web dans le monde utilisent WordPress (source : W3Techs)

  • 🌍 Une communauté active avec des milliers de plugins et thèmes

  • 💼 Utilisé par des grands sites comme The Walt Disney Company, Vogue, ou même la Maison Blanche

✅ Pourquoi choisir WordPress ?

⚙️ Qu’est-ce qu’un CMS ?

Un CMS (Content Management System) est un outil qui permet de créer, gérer et publier du contenu sur un site web sans avoir à écrire le code HTML/CSS à chaque fois.

Avec un CMS comme WordPress, tu peux :

  • Créer une page ou un article depuis un éditeur visuel

  • Ajouter des images, des vidéos, des liens, des menus

  • Modifier l’apparence de ton site sans toucher au code

  • Installer des fonctionnalités supplémentaires via des plugins

🧱 Comment fonctionne WordPress ?

WordPress repose sur une architecture PHP + MySQL :

  • PHP : le langage qui génère les pages dynamiquement

  • MySQL : la base de données qui stocke tous tes contenus, réglages, utilisateurs, etc.

  • HTML/CSS/JS : pour l'affichage dans le navigateur

  • Un thème : définit l'apparence du site

  • Des plugins : ajoutent des fonctionnalités (contact, SEO, e-commerce…)

🧑‍💻 Pour qui est fait WordPress ?

  • Débutants : installation simple, interface intuitive

  • Freelances / agences : développement rapide, maintenance facile

  • Développeurs : extensible via hooks, plugins, thèmes, REST API

  • Entreprises : évolutif, multi-utilisateurs, performant avec un bon hébergement
