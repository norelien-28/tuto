Installer WordPress en local (MAMP, XAMPP, Local)

Introduction

Travailler sur WordPress en local est une étape essentielle pour développer, tester et personnaliser un site sans impacter un site en production. Plusieurs outils permettent d’installer un environnement de serveur local facilement : MAMP, XAMPP ou encore Local by Flywheel.

1. Pourquoi installer WordPress en local ?

  • Développer un site WordPress en toute sécurité.

  • Tester des thèmes, plugins, ou mises à jour sans risque.

  • Travailler sans connexion internet.

  • Gain de temps lors du développement.

2. Les prérequis

  • Un ordinateur Windows, macOS ou Linux.

  • Un logiciel de serveur local (MAMP, XAMPP, ou Local).

  • La dernière version de WordPress (disponible sur wordpress.org).

3. Installer WordPress avec MAMP

Étape 1 : Télécharger et installer MAMP

  • Rendez-vous sur le site officiel : https://www.mamp.info/

  • Téléchargez la version gratuite adaptée à votre OS.

  • Installez MAMP en suivant les instructions.

Étape 2 : Démarrer le serveur

  • Lancez MAMP.

  • Cliquez sur Start Servers pour démarrer Apache et MySQL.

Étape 3 : Créer une base de données

  • Accédez à http://localhost/phpmyadmin dans votre navigateur.

  • Cliquez sur Bases de données, créez une nouvelle base, par exemple wordpress_local.

Étape 4 : Télécharger et installer WordPress

  • Décompressez l’archive WordPress dans le dossier htdocs de MAMP (ex: /Applications/MAMP/htdocs/wordpress).

  • Accédez à http://localhost/wordpress.

  • Suivez l’assistant d’installation : renseignez la base de données (wordpress_local), utilisateur (root), mot de passe (root par défaut avec MAMP).

  • Complétez les informations du site (titre, admin, mot de passe).

4. Installer WordPress avec XAMPP

Étape 1 : Télécharger et installer XAMPP

  • Rendez-vous sur https://www.apachefriends.org/index.html

  • Téléchargez et installez XAMPP pour votre OS.

Étape 2 : Démarrer le serveur

  • Lancez XAMPP Control Panel.

  • Démarrez Apache et MySQL.

Étape 3 : Créer une base de données

  • Ouvrez http://localhost/phpmyadmin.

  • Créez une base wordpress_local.

Étape 4 : Installer WordPress

  • Placez les fichiers WordPress dans xampp/htdocs/wordpress.

  • Visitez http://localhost/wordpress pour lancer l’installation.

  • Configurez la connexion à la base de données (utilisateur root, pas de mot de passe par défaut).

5. Installer WordPress avec Local by Flywheel

Étape 1 : Télécharger Local

  • Téléchargez Local sur https://localwp.com/

  • Installez-le comme une application classique.

Étape 2 : Créer un nouveau site

  • Ouvrez Local, cliquez sur Create a new site.

  • Donnez un nom au site.

  • Choisissez un environnement (recommandé : Preferred).

  • Indiquez vos identifiants admin WordPress.

Étape 3 : Utiliser votre site

  • Local configure automatiquement le serveur, la base de données, PHP, MySQL.

  • Cliquez sur Open Site ou Admin pour accéder à votre site ou tableau de bord WordPress.

6. Conseils et bonnes pratiques

  • Toujours utiliser une version récente de PHP compatible avec WordPress.

  • Ne pas oublier de sécuriser votre environnement local (notamment si accessible en réseau).

  • Sauvegarder régulièrement vos bases et fichiers.

  • Tester les plugins et thèmes avant de les déployer en production.
