08.05 - Les fichiers de base : header, footer, index

Introduction

Dans un thème WordPress, certains fichiers sont indispensables pour structurer correctement les pages. Les trois fichiers de base sont :

  • header.php : contient la partie supérieure HTML de toutes les pages (en-tête).

  • footer.php : contient la partie inférieure HTML de toutes les pages (pied de page).

  • index.php : fichier principal qui sert de modèle pour afficher le contenu.

1. header.php

Ce fichier contient tout ce qui doit apparaître au début de chaque page, notamment :

  • La déclaration DOCTYPE et la balise <html>.

  • L’en-tête <head> avec les métadonnées, le titre, les feuilles de styles et les scripts.

  • Le début de la balise <body> et souvent l’en-tête visuel du site (logo, menu, titre…).

Exemple minimal :

php
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head>
  <meta charset="<?php bloginfo( 'charset' ); ?>">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title><?php wp_title('|', true, 'right'); ?></title>
  <?php wp_head(); ?>
</head>
<body <?php body_class(); ?>>
<header>
  <h1><a href="<?php echo home_url(); ?>"><?php bloginfo('name'); ?></a></h1>
  <nav>
    <?php wp_nav_menu(array('theme_location' => 'main-menu')); ?>
  </nav>
</header>

2. footer.php

Ce fichier contient tout ce qui doit apparaître en bas de chaque page :

  • Le pied de page (copyright, liens, informations).

  • La fermeture de la balise <body> et <html>.

  • L’appel à wp_footer() qui permet d’insérer des scripts avant la fin du corps.

Exemple minimal :

php
<footer>
  <p>&copy; <?php echo date('Y'); ?> <?php bloginfo('name'); ?></p>
</footer>
<?php wp_footer(); ?>
</body>
</html>

3. index.php

Ce fichier sert de modèle par défaut pour afficher le contenu de votre site. Il récupère et affiche les articles ou pages en fonction des requêtes.

Exemple simple :

php
<?php get_header(); ?>

<main>
  <?php if (have_posts()) : ?>
    <?php while (have_posts()) : the_post(); ?>
      <article>
        <h2><a href="<?php the_permalink(); ?>"><?php the_title(); ?></a></h2>
        <div><?php the_excerpt(); ?></div>
      </article>
    <?php endwhile; ?>
  <?php else : ?>
    <p>Aucun contenu disponible.</p>
  <?php endif; ?>
</main>

<?php get_footer(); ?>

4. Remarques importantes

  • Ces fichiers doivent être placés à la racine du dossier de votre thème.

  • Le fichier header.php est appelé par la fonction get_header().

  • Le fichier footer.php est appelé par la fonction get_footer().

  • index.php est obligatoire dans un thème WordPress même s’il sert de fallback.

  • Pour personnaliser la structure de pages spécifiques, il faudra créer d’autres fichiers modèles (single.php, page.php, etc.).

Conclusion

Ces fichiers forment la base indispensable à tout thème WordPress. En maîtrisant leur rôle et leur contenu, tu pourras construire des thèmes robustes et personnalisés.
