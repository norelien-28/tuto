08.04 - Créer un thème WordPress personnalisé

Introduction

Créer un thème WordPress personnalisé permet de contrôler totalement l’apparence et le comportement d’un site. Cela consiste à créer un dossier contenant les fichiers nécessaires qui définissent la structure, le style et les fonctionnalités du thème.

1. Créer le dossier du thème

  • Dans le dossier wp-content/themes/, crée un nouveau dossier pour ton thème.
    Par exemple : mon-theme-personnalise

2. Les fichiers indispensables

Pour qu’un thème soit reconnu par WordPress, il doit contenir au minimum :

2.1 style.css

Ce fichier contient les styles CSS du thème mais aussi une en-tête obligatoire que WordPress utilise pour identifier le thème.

Exemple d’en-tête à placer en début de fichier :

yaml
/*
Theme Name: Mon Thème Personnalisé
Theme URI: http://exemple.com/mon-theme
Author: Ton Nom
Author URI: http://exemple.com
Description: Un thème WordPress personnalisé simple.
Version: 1.0
License: GNU General Public License v2 or later
License URI: http://www.gnu.org/licenses/gpl-2.0.html
Text Domain: mon-theme-personnalise
*/

Ensuite tu peux ajouter tes règles CSS.

2.2 index.php

C’est le fichier principal du thème, servant de fallback pour toutes les pages.

Exemple très simple :

php
<?php get_header(); ?>

<main>
  <h1><?php bloginfo('name'); ?></h1>
  <p><?php bloginfo('description'); ?></p>

  <?php
  if ( have_posts() ) :
    while ( have_posts() ) : the_post();
      the_title('<h2>', '</h2>');
      the_content();
    endwhile;
  else :
    echo '<p>Aucun contenu trouvé</p>';
  endif;
  ?>
</main>

<?php get_footer(); ?>

3. Ajouter les fichiers header.php et footer.php

Pour structurer le thème, on crée ces deux fichiers qui contiennent respectivement le haut et le bas des pages.

3.1 header.php

Exemple minimal :

php
<!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head>
  <meta charset="<?php bloginfo( 'charset' ); ?>">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title><?php wp_title('|', true, 'right'); ?></title>
  <?php wp_head(); ?>
</head>
<body <?php body_class(); ?>>
<header>
  <h1><a href="<?php echo home_url(); ?>"><?php bloginfo('name'); ?></a></h1>
  <p><?php bloginfo('description'); ?></p>
</header>

3.2 footer.php

Exemple minimal :

php
<footer>
  <p>&copy; <?php echo date('Y'); ?> <?php bloginfo('name'); ?></p>
</footer>
<?php wp_footer(); ?>
</body>
</html>

4. Activer le thème

  • Connecte-toi à ton interface d’administration WordPress.

  • Va dans Apparence > Thèmes.

  • Ton thème personnalisé devrait apparaître.

  • Clique sur Activer.

5. Ajouter du style et des fonctionnalités

  • Modifie style.css pour personnaliser l’apparence.

  • Tu peux ajouter un fichier functions.php pour ajouter des fonctionnalités spécifiques.

Conclusion

Créer un thème WordPress personnalisé permet une liberté totale sur le design et le comportement du site. Ce guide couvre la base minimale, à toi d’enrichir ensuite avec des templates supplémentaires, widgets, menus, etc.
