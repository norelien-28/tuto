08.11 – Créer un plugin WordPress simple

Introduction

Créer un plugin WordPress permet d’ajouter des fonctionnalités sans modifier le thème.
C’est la meilleure façon de rendre une fonctionnalité réutilisable, modulaire et indépendante du design.

Même un petit script peut (et devrait souvent) être intégré dans un plugin.

1. Où créer son plugin ?

Tous les plugins se trouvent dans le dossier :

bash
wp-content/plugins/

Tu peux créer un dossier pour ton plugin, par exemple :

bash
wp-content/plugins/mon-plugin/

Et à l’intérieur, un fichier PHP principal :

mon-plugin.php

2. Structure minimale d’un plugin

Voici un exemple de base de plugin :

php
<?php
/**
 * Plugin Name: Mon premier plugin
 * Description: Un exemple simple de plugin WordPress.
 * Version: 1.0
 * Author: Ton Nom
 */

function mon_plugin_message() {
    echo '<p style="padding:1em; background:#f2f2f2;">Bonjour depuis mon plugin !</p>';
}
add_action('admin_notices', 'mon_plugin_message');

Ce plugin affiche un message dans l'administration WordPress.

3. Activer le plugin

  1. Va dans le menu Extensions > Extensions installées

  2. Tu verras apparaître "Mon premier plugin"

  3. Clique sur Activer

Ton plugin est maintenant actif 🎉

4. Ajouter une fonction utile (exemple)

Par exemple, créer un shortcode :

php
function mon_plugin_shortcode_annee() {
    return date('Y');
}
add_shortcode('annee_actuelle', 'mon_plugin_shortcode_annee');

Ensuite, tu peux utiliser [annee_actuelle] dans n’importe quelle page ou article WordPress, et cela affichera l’année en cours.

5. Meilleures pratiques

  • Toujours utiliser un dossier propre pour chaque plugin

  • Préfixer tes fonctions pour éviter les conflits

  • Utiliser des hooks (actions & filtres) pour t’insérer dans WordPress

  • Créer un readme.txt si tu veux le distribuer

6. Et après ?

Une fois à l’aise avec les bases, tu pourras :

  • Ajouter une page de réglages dans l’administration

  • Gérer des options personnalisées

  • Créer des widgets

  • Intégrer l’API REST

  • Charger des scripts ou styles côté public ou admin uniquement

Conclusion

Créer un plugin WordPress simple est à la portée de tous les développeurs.
Cela permet de séparer les fonctionnalités du thème, de réutiliser son code sur plusieurs projets et de gagner en professionnalisme.
