08.13 – Créer et utiliser des shortcodes

Introduction

Les shortcodes permettent d’insérer facilement des éléments dynamiques dans le contenu d’un article, d’une page, ou même d’un widget WordPress, en utilisant une syntaxe simple : 

text
[mon_shortcode]

Ils sont très utiles pour recycler des morceaux de code dans l’éditeur sans avoir à écrire du HTML ou PHP à chaque fois.

1. Créer un shortcode simple

Tu peux créer un shortcode avec la fonction add_shortcode().

Exemple : afficher l’année actuelle

php
function shortcode_annee() {
  return date('Y');
}
add_shortcode('annee', 'shortcode_annee');

➡️ Tu peux maintenant écrire [annee] dans une page, et WordPress affichera 2025.

2. Shortcode avec paramètres (attributs)

Les shortcodes peuvent accepter des paramètres personnalisés, appelés attributs.

Exemple : bouton cliquable

php
function shortcode_bouton($atts) {
  $atts = shortcode_atts([
    'url' => '#',
    'texte' => 'Clique ici'
  ], $atts);

  return '<a class="mon-bouton" href="' . esc_url($atts['url']) . '">' . esc_html($atts['texte']) . '</a>';
}
add_shortcode('bouton', 'shortcode_bouton');

➡️ Utilisation :

text
[bouton url="https://example.com" texte="Découvrir"]

3. Shortcode avec contenu

Tu peux aussi créer un shortcode qui entoure un contenu personnalisé.

Exemple :

php
function shortcode_note($atts, $content = null) {
  return '<div class="note">' . do_shortcode($content) . '</div>';
}
add_shortcode('note', 'shortcode_note');

➡️ Utilisation :

text
[note]Ceci est une note importante.[/note]

4. Où mettre tes shortcodes ?

Tu peux les ajouter :

  • Dans un fichier plugin personnalisé (recommandé)

  • Ou dans le fichier functions.php de ton thème (si c’est spécifique à un thème)

5. Bonnes pratiques

✅ Préfixe toujours le nom de ton shortcode pour éviter les conflits (ex. : mp_bouton)
✅ Utilise esc_html(), esc_url() pour sécuriser l’affichage
✅ Utilise shortcode_atts() pour définir des valeurs par défaut
✅ Ajoute une classe CSS personnalisée si nécessaire

6. Désactiver ou supprimer un shortcode

Pour désactiver un shortcode (utile si un plugin est supprimé mais que le shortcode reste dans le contenu) :

php
remove_shortcode('nom_shortcode');

Ou pour éviter les erreurs :

php
add_shortcode('ancien_shortcode', '__return_empty_string');

Conclusion

Les shortcodes sont un outil simple mais puissant pour ajouter des fonctionnalités réutilisables dans ton contenu WordPress.
Ils permettent de séparer le code de la mise en page, et d’offrir plus de souplesse aux éditeurs de contenu.
