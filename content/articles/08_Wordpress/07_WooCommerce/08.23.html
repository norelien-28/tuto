08.23 – Personnaliser WooCommerce via thèmes et hooks

Introduction

WooCommerce est hautement personnalisable grâce à son intégration dans WordPress et à son système de hooks (actions et filtres). Cela vous permet d’adapter votre boutique à vos besoins sans modifier le cœur du plugin.

1. Personnalisation via les thèmes

1.1 Utiliser un thème compatible WooCommerce

  • De nombreux thèmes WordPress sont spécialement conçus pour WooCommerce.

  • Ces thèmes offrent une mise en page optimisée et des styles adaptés à la boutique.

1.2 Créer un thème enfant pour WooCommerce

  • Pour personnaliser un thème sans perdre les modifications lors des mises à jour, créez un thème enfant.

  • Copiez les fichiers WooCommerce à personnaliser (comme archive-product.php, single-product.php) dans le thème enfant.

  • Modifiez ces fichiers selon vos besoins.

1.3 Modifier les templates WooCommerce

  • WooCommerce utilise des fichiers templates situés dans le dossier woocommerce/templates.

  • En copiant ces fichiers dans votre thème enfant, vous pouvez personnaliser l’apparence des pages produits, panier, paiement, etc.

2. Personnalisation via les hooks (actions et filtres)

2.1 Comprendre les hooks WooCommerce

  • Actions : permettent d’ajouter du contenu ou du code à un emplacement précis.

  • Filtres : permettent de modifier des données avant affichage ou traitement.

2.2 Exemples courants d’utilisation

  • Ajouter un texte personnalisé avant le titre d’un produit.

  • Modifier le bouton “Ajouter au panier”.

  • Modifier les emails envoyés aux clients.

2.3 Comment ajouter un hook

  • Ajoutez du code PHP dans le fichier functions.php de votre thème enfant ou dans un plugin personnalisé. 

php
add_action('woocommerce_before_single_product', 'ma_fonction_personnalisee');
function ma_fonction_personnalisee() {
    echo '<p>Texte personnalisé avant le produit</p>';
}

3. Extensions pour personnalisation avancée

  • Plugins comme WooCommerce Customizer simplifient l’ajout de personnalisations via une interface graphique.

  • Pour des modifications complexes, un développement spécifique est souvent nécessaire.

Conclusion

La personnalisation de WooCommerce via thèmes et hooks vous permet d’adapter votre boutique à votre image et à vos besoins spécifiques, tout en conservant la robustesse du plugin.
