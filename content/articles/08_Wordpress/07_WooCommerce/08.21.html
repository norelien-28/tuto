08.21 – Installer et configurer WooCommerce

Introduction

WooCommerce est la solution e-commerce la plus populaire pour WordPress. Elle permet de transformer votre site en boutique en ligne complète, avec gestion des produits, commandes, paiements, etc.

1. Installation de WooCommerce

1.1 Prérequis

  • Un site WordPress fonctionnel.

  • Accès administrateur au back-office.

1.2 Procédure d’installation

  • Connectez-vous au back-office WordPress.

  • Allez dans Extensions > Ajouter.

  • Recherchez WooCommerce.

  • Cliquez sur Installer maintenant, puis Activer.

2. Configuration initiale

2.1 Assistant de configuration

  • Après activation, un assistant vous guide pour configurer les paramètres de base :

    • Informations sur la boutique (adresse, pays, monnaie).

    • Types de produits à vendre (physiques, numériques).

    • Modes de paiement (PayPal, virement, chèques, etc.).

    • Options de livraison.

2.2 Réglages manuels

  • Vous pouvez modifier ou affiner ces paramètres via WooCommerce > Réglages à tout moment.

3. Ajouter des produits

  • Accédez à Produits > Ajouter.

  • Saisissez le nom, description, prix, images, catégories et autres détails.

  • Configurez les options spécifiques : gestion du stock, poids, dimensions, produits variables.

  • Publiez le produit pour qu’il apparaisse dans la boutique.

4. Autres fonctionnalités importantes

  • Gestion des commandes via WooCommerce > Commandes.

  • Gestion des clients, rapports et statistiques.

  • Extensions WooCommerce pour ajouter des fonctionnalités spécifiques.

Conclusion

WooCommerce est une solution puissante et flexible pour créer une boutique en ligne avec WordPress. Une installation simple et une configuration guidée permettent de rapidement démarrer la vente en ligne.
