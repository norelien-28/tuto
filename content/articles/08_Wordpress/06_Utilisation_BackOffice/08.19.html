08.19 – Utiliser Gutenberg (éditeur de blocs)

Introduction

Gutenberg est l’éditeur par défaut de WordPress depuis la version 5.0. Il remplace l’éditeur classique en proposant une approche basée sur des blocs modulables pour créer facilement du contenu riche et structuré.

1. Qu’est-ce qu’un bloc ?

  • Un bloc est une unité de contenu distincte (paragraphe, image, vidéo, liste, bouton, etc.).

  • Chaque bloc peut être personnalisé indépendamment.

  • Permet de construire des mises en page complexes sans code.

2. Utiliser l’éditeur Gutenberg

2.1 Ajouter un bloc

  • Clique sur le bouton + dans l’éditeur.

  • Choisis le type de bloc à insérer (texte, image, galerie, citation, etc.).

  • Le bloc apparaît dans la page ou l’article.

2.2 Modifier un bloc

  • Clique sur un bloc pour afficher ses options.

  • Personnalise le contenu, la mise en forme, les couleurs, etc.

  • Les options varient selon le type de bloc.

2.3 Déplacer un bloc

  • Utilise les flèches de déplacement situées à gauche du bloc pour le déplacer vers le haut ou le bas.

  • Ou fais un glisser-déposer.

2.4 Supprimer un bloc

  • Sélectionne le bloc puis clique sur les trois points verticaux (…).

  • Choisis Supprimer le bloc.

3. Blocs communs et leur usage

  • Paragraphe : texte simple.

  • Image : insérer une image.

  • Galerie : afficher plusieurs images en grille.

  • Titre : insérer des titres H1, H2, H3, etc.

  • Liste : listes à puces ou numérotées.

  • Bouton : créer un bouton cliquable.

  • Citation : mettre en avant une citation.

  • Colonnes : organiser le contenu en colonnes.

4. Avantages de Gutenberg

  • Interface intuitive et visuelle.

  • Plus de flexibilité pour créer des mises en page.

  • Extension facile avec des blocs personnalisés via plugins.

  • Amélioration continue par la communauté WordPress.

Conclusion

Gutenberg révolutionne la création de contenu sur WordPress en rendant la construction des pages et articles plus simple, modulaire et visuelle. Il est conseillé de se familiariser avec cet éditeur pour tirer parti de toute la puissance de WordPress.
