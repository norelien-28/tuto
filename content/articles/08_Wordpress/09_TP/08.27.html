Kit complet de Travaux Pratiques WordPress

TP1 – Installation et découverte

Objectifs

  • Installer WordPress en local (MAMP, XAMPP, Local)

  • Découvrir l’interface d’administration

  • Créer une page et un article simple

Étapes

  1. Installer un environnement local

    • Télécharge et installe MAMP, XAMPP ou LocalWP.

    • Lance le serveur Apache et MySQL.

  2. Télécharger WordPress

    • Va sur wordpress.org, télécharge la dernière version.

    • Décompresse et copie les fichiers dans le dossier web local (htdocs pour XAMPP, MAMP/htdocs pour MAMP).

  3. Créer une base de données

    • Ouvre phpMyAdmin (ex: localhost/phpmyadmin).

    • Crée une base, par exemple wp_tp1.

  4. Installer WordPress

    • Lance ton navigateur sur localhost/nom_du_dossier_wordpress.

    • Suis le wizard d’installation : renseigne la base de données, utilisateur, mot de passe.

    • Termine l’installation.

  5. Créer une page

    • Connecte-toi au backoffice (localhost/nom_du_dossier_wordpress/wp-admin).

    • Va dans Pages > Ajouter.

    • Crée une page “Bienvenue” avec un texte simple.

    • Publie.

  6. Créer un article

    • Articles > Ajouter.

    • Titre : “Mon premier article”

    • Contenu : Un paragraphe simple.

    • Publie.

TP2 – Création de thème basique

Objectifs

  • Créer un thème enfant

  • Ajouter les fichiers header.php, footer.php et index.php

  • Utiliser The Loop pour afficher les articles

Étapes

  1. Créer un dossier pour le thème enfant

    • Dans wp-content/themes/, crée un dossier mon-theme-enfant.

  2. Créer le fichier style.css 

    css
    /*
Theme Name: Mon thème enfant
Template: twentytwentyone
*/

    Remplace twentytwentyone par le thème parent installé.

  3. Créer le fichier functions.php 

    php
    <?php
function mon_theme_enfant_enqueue_styles() {
    wp_enqueue_style('parent-style', get_template_directory_uri() . '/style.css');
}
add_action('wp_enqueue_scripts', 'mon_theme_enfant_enqueue_styles');
?>

  4. Créer header.php 

    php
    <!DOCTYPE html>
<html <?php language_attributes(); ?>>
<head>
    <meta charset="<?php bloginfo('charset'); ?>">
    <title><?php wp_title('|', true, 'right'); ?></title>
    <?php wp_head(); ?>
</head>
<body <?php body_class(); ?>>
<header>
    <h1><a href="<?php echo home_url(); ?>"><?php bloginfo('name'); ?></a></h1>
    <nav><?php wp_nav_menu(); ?></nav>
</header>

  5. Créer footer.php 

    php
    <footer>
    <p>&copy; <?php echo date('Y'); ?> Mon site WordPress</p>
</footer>
<?php wp_footer(); ?>
</body>
</html>

  6. Créer index.php 

    php
    <?php get_header(); ?>

<main>
<?php if ( have_posts() ) : ?>
    <?php while ( have_posts() ) : the_post(); ?>
        <article>
            <h2><a href="<?php the_permalink(); ?>"><?php the_title(); ?></a></h2>
            <div><?php the_excerpt(); ?></div>
        </article>
    <?php endwhile; ?>
<?php else : ?>
    <p>Aucun article trouvé.</p>
<?php endif; ?>
</main>

<?php get_footer(); ?>

  7. Activer le thème

    • Dans l’admin WordPress > Apparence > Thèmes

    • Active “Mon thème enfant”

TP3 – Contenu personnalisé (Custom Post Type)

Objectifs

  • Créer un CPT “Portfolio”

  • Ajouter des champs personnalisés avec Advanced Custom Fields (ACF)

  • Créer un template pour afficher le CPT

Étapes

  1. Créer le CPT dans functions.php du thème enfant 

    php
    function mon_theme_register_cpt() {
    $args = array(
        'labels' => array(
            'name' => 'Portfolios',
            'singular_name' => 'Portfolio',
        ),
        'public' => true,
        'has_archive' => true,
        'supports' => array('title', 'editor', 'thumbnail'),
    );
    register_post_type('portfolio', $args);
}
add_action('init', 'mon_theme_register_cpt');

  2. Installer et activer le plugin ACF

    • Dans admin WP > Extensions > Ajouter > Cherche “Advanced Custom Fields” > Installer > Activer.

  3. Créer des champs personnalisés

    • Admin WP > Champs personnalisés > Ajouter un groupe.

    • Par exemple : “Lien projet”, “Date de réalisation”, “Technologies utilisées”.

  4. Créer un template pour le CPT single-portfolio.php dans le thème enfant 

    php
    <?php get_header(); ?>

<main>
    <?php if ( have_posts() ) : while ( have_posts() ) : the_post(); ?>
        <article>
            <h1><?php the_title(); ?></h1>
            <div><?php the_content(); ?></div>
            <p>Lien : <a href="<?php the_field('lien_projet'); ?>"><?php the_field('lien_projet'); ?></a></p>
            <p>Date : <?php the_field('date_de_realisation'); ?></p>
            <p>Technologies : <?php the_field('technologies_utilisees'); ?></p>
        </article>
    <?php endwhile; endif; ?>
</main>

<?php get_footer(); ?>

  5. Créer une archive archive-portfolio.php si besoin

TP4 – Plugins et fonctionnalités

Objectifs

  • Créer un plugin simple

  • Ajouter un shortcode

  • Utiliser un hook (action ou filtre)

Étapes

  1. Créer un dossier dans wp-content/plugins/mon-plugin

  2. Créer un fichier mon-plugin.php 

    php
    <?php
/*
Plugin Name: Mon Plugin Simple
Description: Plugin de test pour apprendre les bases.
Version: 1.0
Author: Moi
*/

// Shortcode [bonjour]
function mp_bonjour_shortcode() {
    return "<p>Bonjour, bienvenue sur mon site !</p>";
}
add_shortcode('bonjour', 'mp_bonjour_shortcode');

// Ajouter un message après chaque contenu d'article
function mp_ajouter_message($content) {
    if (is_single()) {
        $content .= "<p>Merci de votre lecture !</p>";
    }
    return $content;
}
add_filter('the_content', 'mp_ajouter_message');

  3. Activer le plugin dans l’admin

  4. Tester le shortcode dans une page/post :
    [bonjour]
