2.1.1 – Bases du HTML

Introduction

Le HTML (HyperText Markup Language) est le langage de base utilisé pour créer des pages web. Il permet de structurer le contenu (titres, paragraphes, images, liens, etc.) que les navigateurs affichent.
Ce tutoriel t’apprend les fondations nécessaires pour commencer à créer tes premières pages web.

1. Qu'est-ce que le HTML ?

HTML n'est pas un langage de programmation, mais un langage de balisage. Il utilise des balises (tags) pour indiquer la structure du contenu.

Exemple simple :

html
<h1>Titre principal</h1>
<p>Voici un paragraphe de texte.</p>

2. Structure de base d'une page HTML

Voici le squelette minimal d’un fichier HTML :

html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Ma première page</title>
</head>
<body>
  <h1>Bonjour le monde !</h1>
  <p>Ceci est ma première page HTML.</p>
</body>
</html>

Explication :

Élément Rôle
<!DOCTYPE html> Indique au navigateur qu'on utilise la version HTML5
<html> Racine de la page HTML
<head> Contient les métadonnées : encodage, titre, liens CSS...
<body> Partie visible de la page : tout le contenu que voit l'utilisateur
<title> Titre affiché dans l’onglet du navigateur
