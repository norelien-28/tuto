2.3.2 – Manipulation du DOM (Document Object Model)

Introduction

Le DOM est une interface qui représente la structure d’une page web sous forme d’un arbre d’objets.
Grâce à JavaScript, tu peux accéder, modifier et interagir avec cet arbre pour rendre ta page dynamique.

1. Qu’est-ce que le DOM ?

  • Le DOM transforme le HTML en objets manipulables.

  • Chaque élément HTML devient un nœud dans cet arbre.

  • JavaScript utilise le DOM pour changer le contenu, la structure, ou le style de la page.

2. Sélectionner des éléments

Méthodes courantes :

  • document.getElementById("id") — sélectionne un élément par son ID.

  • document.getElementsByClassName("classe") — sélectionne une collection d’éléments par classe.

  • document.getElementsByTagName("tag") — sélectionne par balise.

  • document.querySelector("sélecteur CSS") — sélectionne le premier élément correspondant.

  • document.querySelectorAll("sélecteur CSS") — sélectionne tous les éléments correspondants.

Exemple :

js
const titre = document.getElementById("titre");
const boutons = document.querySelectorAll(".btn");

3. Modifier le contenu

Changer le texte :

js
element.textContent = "Nouveau texte";

Modifier le HTML interne :

js
element.innerHTML = "<strong>Texte en gras</strong>";

4. Modifier les styles

js
element.style.color = "red";
element.style.backgroundColor = "#f0f0f0";
element.style.margin = "10px";

5. Ajouter ou supprimer des éléments

Créer un nouvel élément

js
const nouveauParagraphe = document.createElement("p");
nouveauParagraphe.textContent = "Je suis un nouveau paragraphe.";

Ajouter à la page

js
document.body.appendChild(nouveauParagraphe);

Supprimer un élément

js
const elementASupprimer = document.getElementById("aSupprimer");
elementASupprimer.remove();

6. Gestion des classes CSS

js
element.classList.add("nouvelle-classe");
element.classList.remove("ancienne-classe");
element.classList.toggle("classe-toggle");

7. Écouter les événements

js
element.addEventListener("click", function() {
  alert("Clic détecté !");
});

8. Exemple complet

html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8" />
  <title>Manipulation DOM</title>
  <style>
    .highlight {
      background-color: yellow;
    }
  </style>
</head>
<body>
  <h1 id="titre">Titre initial</h1>
  <button id="btnChange">Changer le titre</button>

  <script>
    const btn = document.getElementById("btnChange");
    const titre = document.getElementById("titre");

    btn.addEventListener("click", () => {
      titre.textContent = "Titre modifié !";
      titre.classList.toggle("highlight");
    });
  </script>
</body>
</html>

9. Exercices

  1. Crée un bouton qui ajoute un paragraphe avec du texte au clic.

  2. Ajoute un bouton pour supprimer le dernier paragraphe ajouté.

  3. Modifie la couleur d’un élément quand la souris passe dessus.

  4. Change la classe CSS d’un élément au clic.

10. Conclusion

La manipulation du DOM est essentielle pour rendre tes pages interactives.
Avec un bon contrôle du DOM, tu peux créer des interfaces riches et dynamiques.
