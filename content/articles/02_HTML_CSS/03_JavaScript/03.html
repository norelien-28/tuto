2.3.3 – Fonctions avancées en JavaScript

Introduction

Les fonctions sont au cœur de JavaScript.
Après avoir vu les bases, ce tuto approfondit les fonctions avancées : expressions, fléchées, paramètres par défaut, closures, callback…

1. Fonction déclaration vs fonction expression

Fonction déclaration (classique) :

js
function saluer() {
  console.log("Bonjour !");
}
saluer();

Fonction expression :

js
const saluer = function() {
  console.log("Bonjour !");
};
saluer();

2. Fonctions fléchées (arrow functions)

Syntaxe plus courte, avec quelques particularités sur le this.

js
const saluer = () => {
  console.log("Bonjour !");
};

saluer();

Pour une seule instruction :

js
const carre = x => x * x;
console.log(carre(5)); // 25

3. Paramètres par défaut

Tu peux donner une valeur par défaut à un paramètre :

js
function direBonjour(nom = "inconnu") {
  console.log(`Bonjour, ${nom} !`);
}

direBonjour("Alice"); // Bonjour, Alice !
direBonjour();        // Bonjour, inconnu !

4. Paramètres rest et opérateur spread

Paramètres rest : regrouper plusieurs arguments dans un tableau

js
function somme(...nombres) {
  return nombres.reduce((acc, val) => acc + val, 0);
}

console.log(somme(1, 2, 3)); // 6

Opérateur spread : décomposer un tableau en arguments

js
const tab = [1, 2, 3];
console.log(Math.max(...tab)); // 3

5. Closures (fermetures)

Une fonction qui "mémorise" son environnement lexical.

js
function creerCompteur() {
  let count = 0;
  return function() {
    count++;
    console.log(count);
  };
}

const compteur = creerCompteur();
compteur(); // 1
compteur(); // 2

6. Callbacks

Une fonction passée en argument d’une autre fonction.

js
function demander(question, callback) {
  const reponse = prompt(question);
  callback(reponse);
}

demander("Quel est ton nom ?", function(nom) {
  console.log(`Bonjour, ${nom}`);
});

7. Fonctions anonymes et auto-invoquées (IIFE)

js
(function() {
  console.log("Fonction auto-invoquée !");
})();

8. Exercices

  1. Crée une fonction fléchée qui retourne le double d’un nombre.

  2. Écris une fonction avec paramètre par défaut pour saluer quelqu’un.

  3. Crée un compteur avec closure qui affiche le nombre d’appels.

  4. Utilise un callback pour afficher une alerte après un délai.

9. Conclusion

Les fonctions avancées permettent d’écrire un code plus flexible, lisible et puissant.
Comprendre closures et callbacks est indispensable pour maîtriser JavaScript moderne.
