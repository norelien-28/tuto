2.2.2 – Mise en page avancée

Introduction

Avec les bases du CSS, tu peux déjà styliser ta page.
Mais pour organiser précisément les éléments à l’écran (menus, colonnes, cartes, sections...), il te faut comprendre les outils de mise en page.

Dans ce tuto, on explore les techniques modernes de layout en CSS.

1. Le modèle de boîte (Box Model)

Chaque élément HTML est une boîte rectangulaire composée de 4 parties : 

lua
+---------------------------+
|        margin             |
|  +---------------------+  |
|  |      border         |  |
|  |  +---------------+  |  |
|  |  |   padding     |  |  |
|  |  | +-----------+ |  |  |
|  |  | |  content  | |  |  |
|  |  | +-----------+ |  |  |
|  |  +---------------+  |  |
|  +---------------------+  |
+---------------------------+
Terme Rôle
content Le contenu (texte, image…)
padding Espace intérieur entre le contenu et la bordure
border Bordure de l’élément
margin Espace extérieur entre l’élément et les autres

Exemple CSS :

css
div {
  margin: 20px;
  padding: 10px;
  border: 2px solid black;
}

2. Les types de positionnement

a. static (par défaut)

  • Les éléments s’affichent dans l’ordre du document.

b. relative

  • L’élément reste dans le flux mais peut être décalé (avec top, left, etc.) 

css
div {
  position: relative;
  top: 10px;
  left: 20px;
}

c. absolute

  • Positionné par rapport à son parent positionné

  • Sort du flux normal

css
.parent {
  position: relative;
}

.enfant {
  position: absolute;
  top: 0;
  right: 0;
}

d. fixed

  • Fixé par rapport à la fenêtre du navigateur

css
.menu {
  position: fixed;
  top: 0;
  width: 100%;
}

e. sticky

  • Reste dans le flux, mais se fige à un point précis en scrollant

css
.titre {
  position: sticky;
  top: 0;
}

3. Mise en page avec Flexbox

Flexbox permet de créer des mises en page en ligne ou en colonne de façon simple et responsive.

Exemple :

html
<div class="container">
  <div>Bloc 1</div>
  <div>Bloc 2</div>
  <div>Bloc 3</div>
</div>

css
.container {
  display: flex;
  gap: 20px;
}

.container > div {
  background: lightgray;
  padding: 20px;
  flex: 1;
}

Propriétés utiles :

Propriété Effet
display: flex Active Flexbox
flex-direction Axe principal (row ou column)
justify-content Alignement horizontal
align-items Alignement vertical
gap Espace entre les éléments

4. Mise en page avec CSS Grid

Grid est un système en 2 dimensions (lignes + colonnes), très puissant pour créer des mises en page complexes.

Exemple :

html
<div class="grid">
  <div>A</div>
  <div>B</div>
  <div>C</div>
  <div>D</div>
</div>

css
.grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 10px;
}

.grid > div {
  background: #ddd;
  padding: 20px;
}

Propriétés utiles :

Propriété Rôle
display: grid Active Grid
grid-template-columns Définit le nombre et la taille des colonnes
grid-template-rows (optionnel) lignes
gap Espaces entre lignes/colonnes

5. Comparaison : Flexbox vs Grid

Critère Flexbox Grid
Dimension 1D (ligne ou colonne) 2D (ligne + colonne)
Utilisation Alignement, menus, cartes Layouts complexes, grilles
Complexité Simple à moyen Moyen à avancé

6. Exemple combiné

html
<header>...</header>

<main class="grid">
  <aside>Menu</aside>
  <section>Contenu principal</section>
</main>

<footer>© 2025</footer>

css
main.grid {
  display: grid;
  grid-template-columns: 200px 1fr;
  gap: 20px;
}

7. Exercices pratiques

  1. Crée une mise en page avec :

    • Un header (logo + menu horizontal)

    • Un main avec :

      • Une colonne de navigation

      • Une section de contenu

    • Un footer

  2. Implémente deux versions :

    • Une version Flexbox

    • Une version Grid

  3. Ajoute des margin, padding et border pour bien visualiser les zones.

8. Conclusion

La mise en page en CSS est essentielle pour créer des sites lisibles, modernes et responsives. Grâce à Flexbox et Grid, tu peux construire presque toutes les structures que tu imagines.

Points à retenir :

  • Le box model est la base de tout élément HTML.

  • Le positionnement permet un contrôle précis.

  • Flexbox = idéal pour les alignements simples (menus, cartes…)

  • Grid = parfait pour les layouts structurés (grilles, tableaux…)
