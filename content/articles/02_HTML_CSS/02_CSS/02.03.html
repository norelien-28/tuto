2.2.1 – Introduction au CSS

Introduction

Le CSS (Cascading Style Sheets) est le langage utilisé pour définir le style et l’apparence visuelle des pages HTML. Il permet de personnaliser les couleurs, les polices, les marges, la disposition, etc.

Tu as déjà vu le squelette HTML. Le CSS, lui, est ce qui transforme cette structure en une interface visuelle attrayante.

1. À quoi sert le CSS ?

HTML = structure

CSS = apparence

Exemples de ce que CSS permet :

  • Changer la couleur d’un texte

  • Modifier la taille d’une police

  • Ajouter un fond coloré

  • Aligner des éléments

  • Créer des mises en page responsive (adaptées aux écrans)

2. Trois façons d’intégrer du CSS

a. CSS en ligne (à éviter sauf cas spécifiques)

html
<p style="color: red;">Texte rouge</p>

b. CSS interne (dans un bloc <style> dans le HTML) 

html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>

c. CSS externe (méthode recommandée)

  • Fichier style.css :

css
p {
  color: green;
}

  • Dans le HTML :

html
<head>
  <link rel="stylesheet" href="style.css">
</head>

✅ Avantages du CSS externe :

  • Séparation claire entre contenu (HTML) et style (CSS)

  • Réutilisable sur plusieurs pages

  • Plus facile à maintenir

3. Syntaxe de base du CSS

css
/* Sélecteur */ {
  propriété: valeur;
}

Exemple :

css
h1 {
  color: darkblue;
  font-size: 32px;
}

Le sélecteur (ici h1) désigne quel élément on veut styliser.
Chaque propriété (ex. color, font-size) a une valeur.

4. Sélecteurs courants

Sélecteur Cible
h1, p, div Balise HTML
.maClasse Élément avec une classe
#monId Élément avec un id unique
* Tous les éléments
element1, element2 Plusieurs éléments

Exemple :

css
.maClasse {
  background-color: yellow;
}

#monId {
  font-weight: bold;
}

Dans le HTML :

html
<p class="maClasse">Texte avec fond jaune</p>
<p id="monId">Texte en gras</p>

5. Propriétés CSS utiles

Propriété Effet
color Couleur du texte
background-color Couleur de fond
font-size Taille du texte
text-align Alignement du texte (left, center, right)
margin Marge externe
padding Marge interne
border Bordure autour de l’élément
width, height Taille de l’élément

6. Exemple complet

HTML :

html
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Ma page stylée</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1 class="titre-principal">Bienvenue</h1>
  <p>Voici un paragraphe simple.</p>
</body>
</html>

CSS (style.css) :

css
body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

.titre-principal {
  color: #3366cc;
  text-align: center;
}

7. Exercices pratiques

  1. Crée une page HTML avec un fichier CSS externe.

  2. Applique un fond de couleur à la page.

  3. Change la couleur d’un titre <h1>.

  4. Centre un paragraphe avec text-align.

  5. Ajoute une marge autour d’un bloc <div>.

8. Conclusion

Tu viens de découvrir comment fonctionne le CSS, et comment l’associer à tes pages HTML pour améliorer l’apparence de ton site.
Dans les prochains tutos, tu apprendras à aller plus loin avec le positionnement, les boîtes (box model), et les animations.

Points à retenir :

  • CSS contrôle l’apparence du HTML

  • La meilleure méthode est le CSS externe

  • Les sélecteurs permettent de cibler des éléments pour leur appliquer des styles

  • CSS utilise une syntaxe propriété: valeur;
