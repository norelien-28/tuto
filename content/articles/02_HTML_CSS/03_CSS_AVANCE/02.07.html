📄 02.07 — Variables, mixins et fonctions

Cette page couvre trois piliers essentiels du SCSS/Sass qui permettent de créer un code plus dynamique, réutilisable et maintenable :

  • Les variables

  • Les mixins

  • Les fonctions personnalisées

🎯 1. Les variables

Les variables SCSS permettent de stocker des valeurs réutilisables : couleurs, espacements, polices, tailles, etc.

✅ Syntaxe :

scss
$primary-color: #3498db;
$padding-base: 1rem;

button {
  background-color: $primary-color;
  padding: $padding-base;
}

🧠 Bonnes pratiques :

  • Définir les variables dans un fichier dédié : _variables.scss

  • Les organiser par type : couleurs, tailles, etc.

🧰 2. Les mixins

Les mixins sont comme des fonctions CSS réutilisables. On peut leur passer des paramètres, ce qui les rend très puissantes.

✅ Syntaxe :

scss
@mixin button-style($bg-color, $text-color) {
  background-color: $bg-color;
  color: $text-color;
  border: none;
  padding: 0.75rem 1.5rem;
  border-radius: 4px;
}

.btn-primary {
  @include button-style(#3498db, white);
}

.btn-secondary {
  @include button-style(#f1f1f1, #333);
}

🔁 Mixins avec comportement conditionnel :

scss
@mixin flex-center($direction: row) {
  display: flex;
  flex-direction: $direction;
  justify-content: center;
  align-items: center;
}

🧮 3. Les fonctions personnalisées

Sass permet aussi de créer ses propres fonctions, pour effectuer des calculs ou manipuler des valeurs CSS.

✅ Syntaxe :

scss
@function rem($px, $base: 16) {
  @return #{$px / $base}rem;
}

body {
  font-size: rem(18); // 1.125rem
}

💡 Utilise des fonctions pour standardiser les unités ou automatiser des calculs répétitifs.

🧩 Comparatif rapide

Élément Objectif Exemple
Variable Stocker une valeur réutilisable $color: red;
Mixin Réutiliser un bloc de règles @mixin, @include
Fonction Retourner une valeur calculée @function rem($px)

📂 Organisation recommandée

Fichier _mixins.scss :

scss
@mixin clearfix {
  &::after {
    content: "";
    display: table;
    clear: both;
  }
}

Fichier _functions.scss :

scss
@function px-to-rem($px) {
  @return #{$px / 16}rem;
}

Puis dans ton main.scss :

scss
@use 'mixins';
@use 'functions';

📌 Astuce

Tu peux utiliser @debug pour afficher la valeur retournée par une fonction dans la console lors de la compilation : 

scss
@debug rem(32); // Affiche 2rem
