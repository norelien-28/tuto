📄 02.08 — Organisation du code (partials, imports)

L’un des plus grands avantages de Sass/SCSS, c’est la modularité : tu peux découper ton code en plusieurs fichiers pour qu’il soit plus clair, réutilisable et maintenable.

🧱 1. Les partials (_partials.scss)

Un partial (ou fichier partiel) est un fichier SCSS destiné à être importé, et non compilé directement en CSS.

🔸 Convention :

  • On le nomme avec un underscore (_) au début du nom de fichier :

    • ✅ _variables.scss

    • ✅ _mixins.scss

  • Le compilateur ignore ces fichiers sauf s’ils sont importés dans un fichier principal.

📂 2. Structure de projet recommandée

text
scss/
│
├── base/
│   ├── _reset.scss
│   ├── _typography.scss
│
├── components/
│   ├── _buttons.scss
│   ├── _card.scss
│
├── layout/
│   ├── _header.scss
│   ├── _footer.scss
│
├── utils/
│   ├── _variables.scss
│   ├── _mixins.scss
│   ├── _functions.scss
│
└── main.scss   ← Fichier principal à compiler

🔗 3. Importer les fichiers

⚠️ @import est déprécié

Utilise @use ou @forward (nouvelles recommandations Sass).

✅ 4. Utiliser @use

Exemple de base :

scss
// main.scss
@use 'utils/variables';
@use 'base/typography';
@use 'components/buttons';

Cela importe les fichiers et les rend accessibles avec un préfixe (namespace) : 

scss
body {
  color: variables.$text-color;
}

💡 Pour retirer le préfixe :

scss
@use 'utils/variables' as *;

Utiliser avec précaution pour éviter les conflits de noms.

🔁 5. @forward pour redistribuer

@forward permet de centraliser plusieurs fichiers dans un seul.

Exemple :

scss
// utils/_index.scss
@forward 'variables';
@forward 'mixins';
@forward 'functions';

Puis dans main.scss :

scss
@use 'utils/index' as utils;

.selector {
  color: utils.$primary-color;
}

🧠 Résumé des bonnes pratiques

Objectif Outil Sass Exemple
Modulariser le code Partials _buttons.scss
Importer proprement @use @use 'components/buttons'
Organiser plusieurs fichiers @forward @forward 'variables'
Réduire le préfixe as * @use 'variables' as *;

🔧 Compilation finale

Tu ne compiles qu’un seul fichier : le main.scss. 

bash
sass scss/main.scss css/style.css
