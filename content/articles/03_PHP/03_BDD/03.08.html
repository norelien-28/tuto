3.2.8 – Connexion à une base de données MySQL avec PDO en PHP

1. Qu’est-ce que PDO ?

PDO (PHP Data Objects) est une extension PHP qui permet de se connecter à différentes bases de données avec une interface uniforme. Elle offre une meilleure sécurité et flexibilité que l’extension mysqli.

2. Connexion à la base de données

php
$host = 'localhost';
$db   = 'nom_de_la_base';
$user = 'utilisateur';
$pass = 'mot_de_passe';
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";

$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION, // gestion des erreurs
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,       // fetch en tableau associatif
    PDO::ATTR_EMULATE_PREPARES   => false,                  // désactive l’émulation des requêtes préparées
];

try {
    $pdo = new PDO($dsn, $user, $pass, $options);
    echo "Connexion réussie";
} catch (PDOException $e) {
    throw new PDOException($e->getMessage(), (int)$e->getCode());
}

3. Exécuter une requête simple

php
$stmt = $pdo->query('SELECT * FROM utilisateurs');

while ($row = $stmt->fetch()) {
    echo $row['nom'] . "<br>";
}

4. Requêtes préparées (sécurité contre injection SQL)

Préparation et exécution

php
$stmt = $pdo->prepare('SELECT * FROM utilisateurs WHERE email = :email');
$stmt->execute(['email' => 'exemple@mail.com']);
$user = $stmt->fetch();

if ($user) {
    echo $user['nom'];
}

5. Insertion de données

php
$stmt = $pdo->prepare('INSERT INTO utilisateurs (nom, email) VALUES (:nom, :email)');
$stmt->execute([
    'nom' => 'Alice',
    'email' => 'alice@mail.com'
]);

6. Mise à jour et suppression

php
// Mise à jour
$stmt = $pdo->prepare('UPDATE utilisateurs SET nom = :nom WHERE id = :id');
$stmt->execute([
    'nom' => 'Alice Dupont',
    'id' => 1
]);

// Suppression
$stmt = $pdo->prepare('DELETE FROM utilisateurs WHERE id = :id');
$stmt->execute(['id' => 1]);

7. Gestion des erreurs

  • Avec l’option PDO::ATTR_ERRMODE à PDO::ERRMODE_EXCEPTION, les erreurs lancent des exceptions.

  • Toujours utiliser des try/catch pour attraper les exceptions.
