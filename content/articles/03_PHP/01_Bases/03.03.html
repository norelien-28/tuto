3.2.3 – Conditions et boucles en PHP

1. Les conditions

Les conditions permettent d'exécuter du code en fonction d'une expression booléenne.

Syntaxe if

php
if ($age >= 18) {
    echo "Majeur";
} else {
    echo "Mineur";
}

elseif (ou else if)

php
if ($note >= 16) {
    echo "Excellent";
} elseif ($note >= 10) {
    echo "Passable";
} else {
    echo "À revoir";
}

Condition ternaire

Expression courte, très pratique :

php
echo ($age >= 18) ? "Majeur" : "Mineur";

2. Les opérateurs de comparaison

Opérateur Description Exemple
== Égal en valeur $a == $b
=== Égal en valeur et type $a === $b
!= Différent en valeur $a != $b
!== Différent en valeur/type $a !== $b
< Inférieur $a < $b
> Supérieur $a > $b
<= Inférieur ou égal $a <= $b
>= Supérieur ou égal $a >= $b

3. Les boucles

Les boucles permettent de répéter un bloc de code plusieurs fois.

Boucle while

php
$i = 0;
while ($i < 5) {
    echo "Compteur : $i <br>";
    $i++;
}

Boucle do...while

Exécute le bloc au moins une fois avant de tester la condition.

php
$i = 0;
do {
    echo "Compteur : $i <br>";
    $i++;
} while ($i < 5);

Boucle for

php
for ($i = 0; $i < 5; $i++) {
    echo "Compteur : $i <br>";
}

Boucle foreach (pour les tableaux)

php
$fruits = ["pomme", "banane", "cerise"];
foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

4. Interruption de boucle

break

Sort de la boucle immédiatement.

php
for ($i = 0; $i < 10; $i++) {
    if ($i == 3) {
        break;
    }
    echo $i . "<br>";
}

continue

Passe à l’itération suivante.

php
for ($i = 0; $i < 5; $i++) {
    if ($i == 2) {
        continue;
    }
    echo $i . "<br>";
}

5. Exercice simple

Afficher les nombres pairs de 0 à 10 :

php
for ($i = 0; $i <= 10; $i++) {
    if ($i % 2 == 0) {
        echo "$i est pair<br>";
    }
}
