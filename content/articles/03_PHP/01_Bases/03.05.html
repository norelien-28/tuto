3.2.4 – Fonctions en PHP

1. Qu’est-ce qu’une fonction ?

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique. Elle permet de structurer et d’organiser ton code.

2. Déclaration d’une fonction

php
function direBonjour() {
    echo "Bonjour !";
}

Pour appeler la fonction :

php
direBonjour(); // Affiche : Bonjour !

3. Fonctions avec paramètres

Tu peux passer des paramètres à une fonction pour qu’elle reçoive des données.

php
function direBonjourPersonnalise($nom) {
    echo "Bonjour, " . $nom . " !";
}

direBonjourPersonnalise("Alice"); // Affiche : Bonjour, Alice !

4. Valeur de retour

Une fonction peut retourner une valeur avec return.

php
function addition($a, $b) {
    return $a + $b;
}

$resultat = addition(3, 4); // $resultat vaut 7
echo $resultat;

5. Paramètres avec valeur par défaut

Tu peux définir une valeur par défaut si aucun argument n’est passé.

php
function saluer($nom = "invité") {
    echo "Bonjour, " . $nom . " !";
}

saluer(); // Bonjour, invité !
saluer("Paul"); // Bonjour, Paul !

6. Passage par valeur et par référence

  • Par défaut, les arguments sont passés par valeur (copie).

  • Pour passer par référence, utilise &. 

php
function ajouterUn(&$nombre) {
    $nombre++;
}

$a = 5;
ajouterUn($a);
echo $a; // Affiche 6

7. Fonctions anonymes (closures)

Les fonctions peuvent être stockées dans des variables.

php
$saluer = function($nom) {
    echo "Salut, " . $nom;
};

$saluer("Emma"); // Salut, Emma

8. Portée des variables (scope)

Les variables définies à l’intérieur d’une fonction sont locales. 

php
function test() {
    $x = 5;
}

test();
echo $x; // Erreur : $x n’est pas définie ici

Pour accéder à une variable globale, utilise global :

php
$x = 10;

function afficherX() {
    global $x;
    echo $x;
}

afficherX(); // 10
