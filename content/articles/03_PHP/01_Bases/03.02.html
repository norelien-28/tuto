3.2.2 – Syntaxe de base et variables en PHP

🧾 1. Fichiers PHP : structure de base

Un fichier PHP commence généralement par la balise <?php.
Tout code PHP doit être placé à l’intérieur de ces balises. 

php
<?php
  // Ceci est un script PHP
  echo "Bonjour, monde !";
?>

PHP peut être mélangé à du HTML, mais il est recommandé de séparer autant que possible la logique (PHP) de l'affichage (HTML).

🧠 2. Affichage avec echo ou print 

php
echo "Bonjour";
print "Salut";

  • echo est légèrement plus rapide.

  • print retourne toujours 1, donc peut être utilisé dans des expressions.

🪙 3. Déclaration de variables

En PHP, les variables commencent toujours par le symbole $. 

php
$nom = "Alice";
$age = 25;

  • Pas besoin de spécifier le type (string, int, etc.) — PHP est faiblement typé.

  • Les noms de variables sont sensibles à la casse ($nom ≠ $Nom).

📌 4. Types de données courants

Type Exemple
string "Bonjour"
integer 42
float 3.14
boolean true, false
array ["a", "b", "c"]
object (vu plus tard)
null null

🧮 5. Opérateurs

➕ Arithmétiques :

php
$a = 5 + 3;   // 8
$b = 10 % 3;  // 1

🟰 Affectation :

php
$x = 10;
$x += 5; // équivaut à $x = $x + 5;

🤔 Comparaison :

php
$a == $b  // égal en valeur
$a === $b // égal en valeur et en type

⚙️ Logiques :

php
($a > 5 && $b < 10) // ET logique
($a > 5 || $b < 10) // OU logique

🧪 6. Tester une variable

🔍 isset() — La variable existe et n'est pas null 

php
if (isset($nom)) {
  echo "La variable existe";
}

🧼 empty() — La variable est vide (0, "", null, false, etc.) 

php
if (empty($email)) {
  echo "Champ requis";
}

🧹 7. Supprimer une variable

php
unset($maVariable);

🔠 8. Concaténation de chaînes

php
$prenom = "Marie";
echo "Bonjour " . $prenom;

  • On utilise . pour concaténer (et non + comme en JavaScript).

🎯 9. Constantes

php
define("PI", 3.14);
echo PI;

  • Une constante ne commence pas par $.

  • Par convention, on écrit les noms de constantes en majuscules.

🧪 10. Bonus : interpolation de variables

Dans les chaînes entre guillemets doubles, les variables sont interprétées : 

php
$nom = "Paul";
echo "Bonjour $nom"; // Bonjour Paul

Dans les chaînes entre guillemets simples, elles sont littérales : 

php
echo 'Bonjour $nom'; // Bonjour $nom
