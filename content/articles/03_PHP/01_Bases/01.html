3.1.1 – Introduction à PHP

🧾 Qu’est-ce que PHP ?

PHP (Hypertext Preprocessor) est un langage de script côté serveur, principalement utilisé pour créer des sites web dynamiques.
Il est interprété sur le serveur, puis envoie du HTML au navigateur de l'utilisateur.

PHP est intégré à plus de 75 % des sites web dans le monde, notamment via des CMS comme WordPress.

⚙️ Pourquoi apprendre PHP ?

✅ Avantages :

  • Facile à apprendre, syntaxe proche de C/JavaScript.

  • Excellente documentation et grande communauté.

  • Compatible avec tous les serveurs web (Apache, Nginx).

  • Support natif des bases de données (MySQL, PostgreSQL...).

  • Fonctionne bien avec HTML et JavaScript.

💡 Où s’exécute PHP ?

  • Sur le serveur : c’est lui qui traite les fichiers .php.

  • L'utilisateur final ne voit que le résultat HTML.

🔧 Exemple simple :

php
<?php
  $nom = "Jean";
  echo "<h1>Bonjour, $nom !</h1>";
?>

🧩 Résultat dans le navigateur :

html
<h1>Bonjour, Jean !</h1>

📁 Fichiers PHP

  • Extension .php

  • Peuvent contenir un mélange de HTML + PHP

  • Commencent toujours par <?php
    (et se ferment automatiquement à la fin du fichier)

🧠 Syntaxe de base

Variables

php
$nom = "Alice";
$age = 30;

Conditions

php
if ($age >= 18) {
  echo "Majeur";
} else {
  echo "Mineur";
}

Boucles

php
for ($i = 0; $i < 5; $i++) {
  echo "Ligne $i <br>";
}

🌐 Comment exécuter du PHP ?

✅ En local :

  • Utilise un environnement comme XAMPP, MAMP ou Laragon

  • Ou installe manuellement : Apache + PHP + MySQL

✅ En ligne :

  • Hébergeur web supportant PHP (presque tous le font)

⚠️ PHP vs JavaScript

Aspect PHP JavaScript
Exécuté sur Serveur Navigateur (client)
Renvoie HTML HTML / DOM / interactions
Peut accéder à Base de données, fichiers DOM, API du navigateur
Utilisation typique Générer du contenu dynamique côté serveur Gérer l’interaction utilisateur côté client
