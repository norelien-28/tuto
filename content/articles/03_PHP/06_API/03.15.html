3.2.15 – Créer une API REST avec base de données (MySQL/PDO)

1. Introduction

Cette fiche détaille la création d’une API REST en PHP utilisant une base de données MySQL via PDO. Cette approche permet un stockage persistant, des requêtes efficaces et une meilleure gestion des données.

2. Pré-requis

  • Serveur PHP (>=7.0) avec extension PDO activée

  • Serveur MySQL/MariaDB

  • Base de données créée (exemple : ma_db) avec table utilisateurs

3. Structure de la table MySQL

sql
CREATE TABLE utilisateurs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nom VARCHAR(100) NOT NULL
);

4. Connexion à la base de données avec PDO

php
try {
    $pdo = new PDO('mysql:host=localhost;dbname=ma_db;charset=utf8', 'user', 'password');
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
} catch (PDOException $e) {
    http_response_code(500);
    echo json_encode(['message' => 'Erreur de connexion à la base de données']);
    exit;
}

5. Structure du point d’entrée (index.php)

php
<?php
header("Content-Type: application/json");
require 'config.php'; // fichier avec la connexion PDO dans $pdo

$method = $_SERVER['REQUEST_METHOD'];
$uri = $_SERVER['REQUEST_URI'];
$request = explode('/', trim(parse_url($uri, PHP_URL_PATH), '/'));

if ($request[0] !== 'api' || $request[1] !== 'utilisateurs') {
    http_response_code(404);
    echo json_encode(['message' => 'Ressource non trouvée']);
    exit;
}

$id = $request[2] ?? null;

switch ($method) {
    case 'GET':
        if ($id) {
            getUser($id);
        } else {
            getUsers();
        }
        break;

    case 'POST':
        createUser();
        break;

    case 'PUT':
        if ($id) {
            updateUser($id);
        } else {
            http_response_code(400);
            echo json_encode(['message' => 'ID requis pour mise à jour']);
        }
        break;

    case 'DELETE':
        if ($id) {
            deleteUser($id);
        } else {
            http_response_code(400);
            echo json_encode(['message' => 'ID requis pour suppression']);
        }
        break;

    default:
        http_response_code(405);
        echo json_encode(['message' => 'Méthode non autorisée']);
}

6. Fonctions CRUD avec PDO

Lecture des données JSON

php
function getRequestData() {
    return json_decode(file_get_contents('php://input'), true);
}

Récupérer tous les utilisateurs

php
function getUsers() {
    global $pdo;
    $stmt = $pdo->query("SELECT id, nom FROM utilisateurs");
    $users = $stmt->fetchAll(PDO::FETCH_ASSOC);
    echo json_encode($users);
}

Récupérer un utilisateur par ID

php
function getUser($id) {
    global $pdo;
    $stmt = $pdo->prepare("SELECT id, nom FROM utilisateurs WHERE id = ?");
    $stmt->execute([$id]);
    $user = $stmt->fetch(PDO::FETCH_ASSOC);

    if ($user) {
        echo json_encode($user);
    } else {
        http_response_code(404);
        echo json_encode(['message' => 'Utilisateur non trouvé']);
    }
}

Créer un utilisateur

php
function createUser() {
    global $pdo;
    $data = getRequestData();

    if (empty($data['nom'])) {
        http_response_code(400);
        echo json_encode(['message' => 'Le champ "nom" est requis']);
        return;
    }

    $stmt = $pdo->prepare("INSERT INTO utilisateurs (nom) VALUES (?)");
    $stmt->execute([$data['nom']]);

    http_response_code(201);
    echo json_encode(['id' => $pdo->lastInsertId(), 'nom' => $data['nom']]);
}

Mettre à jour un utilisateur

php
function updateUser($id) {
    global $pdo;
    $data = getRequestData();

    if (empty($data['nom'])) {
        http_response_code(400);
        echo json_encode(['message' => 'Le champ "nom" est requis']);
        return;
    }

    $stmt = $pdo->prepare("UPDATE utilisateurs SET nom = ? WHERE id = ?");
    $stmt->execute([$data['nom'], $id]);

    if ($stmt->rowCount() > 0) {
        echo json_encode(['id' => $id, 'nom' => $data['nom']]);
    } else {
        http_response_code(404);
        echo json_encode(['message' => 'Utilisateur non trouvé ou données identiques']);
    }
}

Supprimer un utilisateur

php
function deleteUser($id) {
    global $pdo;
    $stmt = $pdo->prepare("DELETE FROM utilisateurs WHERE id = ?");
    $stmt->execute([$id]);

    if ($stmt->rowCount() > 0) {
        http_response_code(204);
    } else {
        http_response_code(404);
        echo json_encode(['message' => 'Utilisateur non trouvé']);
    }
}

7. Bonnes pratiques

  • Toujours utiliser des requêtes préparées pour éviter les injections SQL

  • Gérer correctement les codes HTTP selon la réponse

  • Valider et nettoyer les données envoyées par le client

  • Séparer la logique métier de la gestion des requêtes (futur refactoring)

  • Implémenter un système d’authentification si l’API est exposée publiquement
