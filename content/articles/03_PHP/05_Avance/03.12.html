3.2.12 – Introduction à la programmation orientée objet (POO) en PHP

1. Qu’est-ce que la POO ?

La programmation orientée objet permet de structurer le code en objets qui regroupent données (propriétés) et comportements (méthodes).

2. Définir une classe et créer un objet

php
class Personne {
    public $nom;

    public function direBonjour() {
        echo "Bonjour, je suis " . $this->nom;
    }
}

// Création d’un objet
$personne = new Personne();
$personne->nom = "Alice";
$personne->direBonjour();

3. Visibilité des membres

Mot-clé Description
public Accessible partout
protected Accessible dans la classe et ses enfants
private Accessible uniquement dans la classe

4. Constructeur

Méthode appelée automatiquement lors de la création de l’objet.

php
class Personne {
    public $nom;

    public function __construct($nom) {
        $this->nom = $nom;
    }
}
$personne = new Personne("Bob");
echo $personne->nom; // Bob

5. Héritage

Une classe peut hériter d’une autre pour réutiliser ses propriétés et méthodes.

php
class Employe extends Personne {
    public $poste;

    public function __construct($nom, $poste) {
        parent::__construct($nom);
        $this->poste = $poste;
    }

    public function travailler() {
        echo $this->nom . " travaille comme " . $this->poste;
    }
}

$employe = new Employe("Claire", "Développeuse");
$employe->travailler();

6. Encapsulation et getters/setters

Protéger les propriétés avec private et utiliser des méthodes pour y accéder. 

php
class CompteBancaire {
    private $solde = 0;

    public function getSolde() {
        return $this->solde;
    }

    public function deposer($montant) {
        if ($montant > 0) {
            $this->solde += $montant;
        }
    }
}

$compte = new CompteBancaire();
$compte->deposer(100);
echo $compte->getSolde(); // 100

7. Autres notions importantes

  • Méthodes statiques : méthodes accessibles sans instance (self::).

  • Interfaces : définissent des méthodes sans implémentation.

  • Traits : permettent de réutiliser du code dans plusieurs classes.
